2. Open markdown file in Chrome
3. Press Ctrl+P (Print)
4. Save as PDF

OPTION 3: Install XeLaTeX (For Future Use)
------------------------------------------
XeLaTeX handles Unicode characters better:

sudo apt-get install texlive-xetex

Then regenerate PDFs with:
pandoc FILE.md -o FILE.pdf --pdf-engine=xelatex --toc
